# Problema 4: Diseñe un programa para calcular el valor final de un producto según su categoría.

## Requisitos:

- [x] Definir una constante IVA = 0.19.

- [x] Solicitar el precio base del producto.

- [x] Si el precio ≤ 0, mostrar error y finalizar.

- [x] Solicitar el código de categoría:

  - Producto básico: no paga IVA.

  - Producto estándar: paga IVA del 19%.

  - Producto de lujo: paga IVA del 19% + recargo del 5%.

- [x] Si el código es inválido, mostrar error y finalizar.

- [x] Calcular y mostrar el valor final según la categoría.
